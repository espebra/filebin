About

This service is provided without any guarantees.

Technical limitations

There are no practical limitations regarding the size and number of files to upload in each tag. This does not mean that there is unlimited amount of storage available.

Content limitations

Do not upload any files that somehow may be considered illegal. This service may shut down if files considered to be illegal are being distributed.

Reporting tags

If you stumble upon files that you think should not be available through this service, please click at the report link and provide a proper explaination.

Security

The files are not being encrypted when stored server side. If you need encryption, you will have to handle the encryption and decryption on the client side using Truecrypt or similar software.

Requirements

You will need a browser with File API support to upload files using this service. You will be able to download files using any browser.

API features

Uploading files

To upload a file, simply issue a POST request to http://{{ host }}/uploader with the headers x-filename, x-size (in bytes), x-tag and x-checksum (md5 checksum) set. The body of the request should contain the contents of the file.

Listing contents of a tag

Tags can be listed in json format at http://{{ host }}/tag?v=json or as a list at http://{{ host }}/tag?v=playlist.

Downloading files

Files can be downloaded at http://{{ host }}/tag/filename.
